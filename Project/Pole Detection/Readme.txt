The code can be run in anaconda prompt.

The following are the instructions :
1) Create a new enivronment (Refer this link: https://uoa-eresearch.github.io/eresearch-cookbook/recipe/2014/11/20/conda/)
2) install pcl (Refer this link: http://pointclouds.org/documentation/tutorials/using_pcl_pcl_config.php)
3) install geo2cart using the command "pip install geo2cart".
4) install numpy and matplotlib (Refer this link: https://www.southampton.ac.uk/~fangohr/blog/installation-of-python-spyder-numpy-sympy-scipy-pytest-matplotlib-via-anaconda.html)
5) install mpl_toolkits (Refer this link: https://github.com/matplotlib/matplotlib/issues/4546)


To run the code

"python PoleDetection.py"


Output Image Generated:
The output image is a 3D image which can be tilted in any direction. The detected poles can be seen in the 3D image.
